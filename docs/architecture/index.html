Architecture

Open Volunteer can be deployed to any cloud provider thanks to automation for building docker images

#Reference deployment

Open Volunteer Platform is being tested and deployed to Red Hat Managed Integration

Organizations can base of simplicity of the deployment and management thanks to Red Hat Managed Integration managed services like:

  • Red Hat SSO (Keycloak Authentication)
  • Red Hat AMQ Online (MQTT protocol broker)
  • Red Hat Code Ready (Clound Native Development Environments)
  • Red Hat OpenShift Dedicated (base layer for running services)
  • MongoDB and Postgress data providers
